Overview

Represents errors that occur during application execution.

Exception and it's descendants are used to communicate between raise and rescue statements in begin ... end blocks. Exception objects carry information about the exception – its type (the exception’s class name), an optional descriptive string, and optional traceback information. Exception subclasses may add additional information.
