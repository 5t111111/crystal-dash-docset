def linger=(val : Int | Nil)

WARNING: The behavior of SO_LINGER is platform specific. Bad things may happen especially with nonblocking sockets. See https://www.nybek.com/blog/2015/04/29/so_linger-on-non-blocking-sockets/ for more information.

nil => disable SO_LINGER Int => enable SO_LINGER and set timeout to Int seconds.

0 => abort on close (socket buffer is discarded and RST sent to peer). Depends on platform and whether shutdown() was called first. >=1 => abort after Num seconds on close. Linux and Cygwin may block on close.
