def self.hex_digest(&block) : String

Yields a context object with an #update(data : String | Slice(UInt8)) method available. Returns a String the hexadecimal representation of the MD5 hash all data passed in.

Crypto::MD5.hex_digest do |ctx|
   ctx.update "f"
   ctx.update "oo"
end                            # => "acbd18db4cc2f85cedef654fccc4a4d8"
