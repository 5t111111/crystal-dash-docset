Overview

An IO that reads and writes from a buffer in memory.

The internal buffer can be resizeable and/or writeable depending on how a MemoryIO is constructed.

def self.new(string : String)

Creates a MemoryIO whose contents are the exact contents of string. The created MemoryIO is non-resizeable and non-writeable.

The IO starts at position zero for reading.

io = MemoryIO.new "hello"
io.pos        # => 0
io.gets(2)    # => "he"
io.print "hi" # raises

def self.new(slice : Slice(UInt8), writeable = true)

Creates a MemoryIO that will read, and optionally write, from/to the given slice. The created MemoryIO is non-resizeable.

The IO starts at position zero for reading.

slice = Slice.new(6) { |i| ('a'.ord + i).to_u8 }
io = MemoryIO.new slice, writeable: false
io.pos  # => 0
io.read # => "abcdef"
